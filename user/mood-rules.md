# 心情-音乐匹配规则

## 心情分类与推荐

### 开心 / Happy
- **推荐风格**: 流行、轻摇滚、电子
- **节奏**: 中等偏快 (120-140 BPM)
- **艺术家示例**: 周杰伦、Maroon 5、Pharrell Williams
- **推荐语**: "心情不错！来首更开心的歌延续这份快乐！"

### 平静 / Calm
- **推荐风格**: 民谣、轻音乐、钢琴曲
- **节奏**: 缓慢 (60-80 BPM)
- **艺术家示例**: 陈鸿宇、Yiruma、久石让
- **推荐语**: "享受这份宁静..."

### 焦虑 / Anxious
- **推荐风格**: 纯音乐、氛围音乐、古典
- **节奏**: 平稳 (70-90 BPM)
- **艺术家示例**: 坂本龙一、Max Richter
- **推荐语**: "深呼吸，让音乐带你平静下来..."

### 疲惫 / Tired
- **推荐风格**: Lo-Fi、爵士、轻电子
- **节奏**: 舒缓 (80-100 BPM)
- **艺术家示例**: Nujabes、Norah Jones
- **推荐语**: "辛苦了，来首舒缓的歌放松一下..."

### 专注 / Focus
- **推荐风格**: 纯音乐、古典、环境音
- **节奏**: 平稳无强烈起伏
- **艺术家示例**: Ludovico Einaudi、电影原声
- **推荐语**: "专注时刻，不打扰的音乐陪伴你..."

### 浪漫 / Romantic
- **推荐风格**: R&B、抒情流行、爵士
- **节奏**: 缓慢流畅
- **艺术家示例**: 陈奕迅、方大同、John Legend
- **推荐语**: "这个时刻，适合这样的旋律..."

### 怀旧 / Nostalgic
- **推荐风格**: 经典老歌、校园民谣
- **年代**: 2000s-2010s
- **艺术家示例**: 五月天、苏打绿、老狼
- **推荐语**: "这首歌，会不会让你想起什么..."

## 天气匹配

### 晴天
- 活力、明快的音乐
- 适合户外、开车听的歌

### 雨天
- 抒情、氛围感强的音乐
- 钢琴曲、民谣

### 阴天
- 温暖、治愈的音乐
- 轻摇滚、R&B

### 雪天
- 安静、纯净的音乐
- 古典、轻音乐

## 时间匹配

- **早晨**: 清新、活力的音乐，帮助唤醒
- **深夜**: 安静、不打扰的音乐，陪伴入眠
- **深夜加班**: 提神但不焦虑的音乐
